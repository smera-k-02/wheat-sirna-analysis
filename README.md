# wheat-sirna-analysis
siRNA expression profiling in wheat under leaf rust infection
## Objectives
- Identify siRNA clusters
- Perform differential expression
- Predict targets

## Tools Used
FastQC, Cutadapt, Bowtie2, ShortStack, OmicsBox, psRNA Target

## Workflow
1. Quality control
2. Trimming
3. Mapping
4. Cluster identification
5. Differential expression
6. Target prediction
7. Functional Annotation
